# WPAstra Website Templates – QA Testing Report

## About WPAstra

WPAstra is a WordPress platform that offers ready-made website templates and themes for various industries and use cases. Users can browse, preview, filter, and import templates to quickly build professional websites. This testing assignment focuses on evaluating the Website Templates page for functionality, responsiveness, usability, and UI consistency.

## Project Overview

This repository contains the results of a manual testing assignment conducted on the WPAstra Website Templates page:

https://wpastra.com/website-templates/

The objective was to evaluate the website's functionality, responsiveness, usability, UI consistency, and overall user experience across different screen sizes.

---

## Testing Scope

The following areas were tested:

- Navigation menus and dropdowns
- Template cards and interactions
- Preview and Details functionality
- Filtering and sorting features
- Responsive behavior (Desktop, Tablet, Mobile)
- UI consistency and layout
- Tooltip behavior
- Navigation usability
- Content presentation

## Summary

A total of **15 issues** were identified during testing.

The reported issues include:

- UI inconsistencies
- Responsive design issues
- Navigation usability problems
- Duplicate content display
- Layout and spacing defects
- Interaction inconsistencies

## Key Findings

- Inconsistent card interaction behavior between pages
- Duplicate templates appearing in filtered results
- Mobile responsiveness issues affecting buttons and navigation
- Layout inconsistencies between Home and Details pages
- Navigation dropdown usability concerns
- Tooltip styling inconsistencies
- Missing filter reset functionality

---

## Conclusion

The website is generally functional; however, several usability, UI consistency, and responsive design issues were identified that may impact the overall user experience. Addressing these findings would improve consistency, accessibility, and ease of navigation across devices.
